# 📚 Classroom Booking System

## 📌 Overview  
This project is a **Database Management System** developed as part of an academic assignment by:

- Anudeep Reddy  
- Sanjeet Roy  
- Soham Gupta  

The system is designed to efficiently **store, manage, and retrieve classroom booking data** using structured database techniques. It demonstrates core concepts of database design, querying, and application integration.

---

## 🚀 Features  
- ✅ Add new records  
- ✅ View existing data  
- ✅ Update records  
- ✅ Delete records  
- ✅ Search functionality  
- ✅ Structured database queries
